# Aplikasi-Login
Sebuah aplikasi dengan fitur login.

Alur Fitur Login:
1. Memasukkan `email` dan `password`
    - `email    :` <b>admin@dicoding.com</b>
    - `password :` <b>superpassword</b>
3. Memvalidasi masukan.
4. Menampilkan halaman home.
